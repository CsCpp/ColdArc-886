# 🛠 ColdArc-886: TIG & Cold Welding Controller

## 📖 Project Overview / Обзор проекта

### English
This project provides a robust, highly EMI-resistant firmware solution for DIY TIG welding machines based on the **PIC16F886** microcontroller. It transforms a basic board into a professional-grade controller capable of managing gas flow, high-frequency (HF) arc ignition, and complex welding sequences (including Cold Weld).

### Русский
Этот проект представляет собой надежную, защищенную от высокочастотных помех прошивку для самодельных сварочных аппаратов TIG на базе микроконтроллера **PIC16F886**. Прошивка превращает базовую плату в профессиональный контроллер, управляющий подачей газа, осциллятором поджига (HF) и сложными циклами сварки (включая режим холодной сварки / Cold Weld).

---

## ⚡ Key Features / Основной функционал

### English
* **4 Operating Modes:** 2T, 4T (Pro logic: press-release to weld, press to stop), Cold Pulse, and Tack (Spot).
* **Precise Timing:** Configurable Pre-flow, Post-flow, and HF arc times.
* **Extreme EMI Hardening:** Software filters (160ms debounce) and specific MCU fuses (`MCLRE=OFF`, `BOREN=OFF`) prevent crashes from HF noise.
* **Custom UI:** Adjustable parameters via an intuitive LED bar scale.
* **Safety First:** Prevents arc ignition if the torch button is held during power-up.

### Русский
* **4 режима работы:** 2Т, 4Т (Про-логика: нажал/отпустил = варим, нажал = стоп), Cold Pulse (холодная сварка) и Tack (прихватки).
* **Точные тайминги:** Настраиваемая длительность предгаза, постгаза и работы осциллятора.
* **Экстремальная защита от ВЧ-помех (EMI):** Цифровые фильтры (160 мс) и отключение уязвимых модулей чипа (`MCLRE=OFF`, `BOREN=OFF`) предотвращают зависания от осциллятора.
* **Интерфейс:** Регулировка параметров через визуальную светодиодную шкалу.
* **Безопасность:** Блокировка старта, если кнопка горелки зажата при включении аппарата.

## 🛡️ Hardware Hardening / Аппаратная защита (CRITICAL)

**English:** To survive the massive electromagnetic interference (EMI) generated by the HF oscillator, your hardware **must** include the following:
1. **Opto-isolation:** Use optocouplers (e.g., PC817) or relays between the MCU and the torch button/power relays.
2. **Flyback Diodes:** Install `1N4007` diodes in parallel with all 12V relay coils (cathode to +12V).
3. **Snubber Caps:** Place `100nF` (0.1µF) ceramic capacitors across relay contacts and across the MCU's VDD/VSS pins.
4. **Star Grounding:** Connect the MCU's ground (GND) to the metal chassis of the welder in exactly **one** point. Do not connect the 12V relay ground to the MCU ground if using optocouplers.

**Русский:** Для выживания контроллера рядом с мощным ВЧ-осциллятором, аппаратная часть **должна** включать:
1. **Развязку:** Используйте оптопары (PC817) или промежуточные реле для кнопки горелки и силовых выходов.
2. **Диоды:** Обязательно установите защитные диоды (`1N4007`) параллельно всем катушкам реле 12В (катодом к плюсу).
3. **Конденсаторы:** Припаяйте керамику `0.1 мкФ` (104) на контакты кнопки возле платы и на пины питания самого МК.
4. **Заземление:** Соедините минус МК с металлическим корпусом аппарата в **одной** точке. Если используете оптопары, не объединяйте минус реле 12В с минусом МК.

## 🎛️ Operating Modes & Setup / Управление и настройка

### 1. Modes (Press `UP` / Режимы работы)
* **1 LED:** `2T` (Standard) — Weld while the button is held. / Варим, пока держим кнопку.
* **2 LED:** `4T` (Trigger) — Press & release to start arc. Press to stop arc. / Нажал/отпустил = старт, нажал = стоп.
* **3 LED:** `Cold Weld` — Series of highly precise short pulses. / Серия коротких импульсов (холодная сварка).
* **4 LED:** `Tack` (Spot) — Single precise pulse. / Одиночный точный импульс (прихватка).
* **Hold `DOWN`:** Manual gas purge. / Ручная продувка газа.

### 2. Settings Menu / Настройки параметров
Hold **UP + DOWN** for 1 second to enter the menu. Press the **Torch Button (START)** to cycle through parameters. Use `UP`/`DOWN` to change values.

| Active LED (Blinking) | Parameter / Параметр | Range (1-8 scale) |
| :---: | :--- | :--- |
| **LED 1** | Pre-Gas / Предгаз | 0.25 – 2.0 sec |
| **LED 2** | Post-Gas / Постгаз | 0.375 – 3.0 sec |
| **LED 3** | Arc Time / Время HF | 0.1 – 0.8 sec * |
| **LED 4** | Tack Time / Точка | 0.1 – 0.8 sec * |

> **⚡️ COLD WELD Exceptions (LED 3 Mode):** > In Cold Weld mode, the last two parameters behave differently for extreme precision:
> * **LED 3 (Arc):** Flash duration / Длительность вспышки (10 – 80 ms).
> * **LED 4 (Pulse):** Pause duration / Длительность паузы (50 – 400 ms).

## 🚀 Programming Guide / Прошивка

1. **Setup:** Install **MPLAB X IDE** and **XC8 Compiler**.
2. **Project:** Create a "Standalone Project" for **PIC16F886** and add `main.c`.
3. **Important Fuses:** The code includes specific pragmas. Do not change them. 
   `#pragma config MCLRE = OFF` and `#pragma config BOREN = OFF` are crucial for HF stability.
4. **Flash:** In project properties under your programmer (e.g., PICkit 3), enable "Power target circuit" (5.0V). Click "Make and Program Device".

## 🛠 Troubleshooting / Частые проблемы

* **LED 7/8 issue:** Expected behavior; bits 6 and 7 of `PORTA` are swapped in software to make routing the PCB easier.
* **MCU Resets during HF ignition:** 1. Verify you actually flashed the chip with `MCLRE = OFF` and `BOREN = OFF`. 
  2. Ensure you have flyback diodes on all mechanical relays.
  3. Check your Star Ground connection.
  *Note: A 10k pull-up resistor on the MCLR pin is no longer required as the pin is disabled internally for EMI protection.*
* **Spontaneous Arc starts:** The torch button wires are acting as an antenna. Add a 1kΩ external pull-up resistor to VDD on pin `RB3` and pass the torch wires through a ferrite bead.
